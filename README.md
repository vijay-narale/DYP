# 🚀 PlaceIQ - AI Career Co-Pilot

PlaceIQ is an advanced AI-driven platform designed to bridge the gap between candidate skillsets and industry requirements.It leverages Large Language Models (LLMs) to provide actionable insights for career growth and recruitment.

---

## 🌟 Overview

PlaceIQ acts as an intelligent career consultant, helping users analyze their professional profiles, identify skill gaps, and prepare for interviews through data-driven AI models.

### **✨ Core Features**

- **📄 Smart Resume Parsing**: Extracts structured data from PDF resumes using AI-powered OCR and parsing logic.
- **🔍 Profile Analysis**: Deep-dive assessment of strengths, weaknesses, and overall market relevance.
- **🛣️ Career Roadmap Generator**: Step-by-step guidance to achieve target roles or transitions.
- **🎯 Skill Gap Analysis**: Compares candidate profiles with specific Job Descriptions (JDs) to identify missing competencies.
- **🎙️ Interview Preparation**: Generates tailored technical and behavioral questions based on the candidate's profile.
- **📚 JD Management**: A library of job descriptions for better market alignment and profile tailoring.

---

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, Multer (File Handling), PDF-parse.
- **Database / Auth**: Supabase.
- **AI Engine**: Anthropic (Claude), GPT (via OpenRouter).

---

## 📂 Project Structure

- `/frontend`: User interface and client-side logic.
- `/backend`: API routes, AI integrations, and middleware.
- `/supabase`: SQL migrations and database configuration.
